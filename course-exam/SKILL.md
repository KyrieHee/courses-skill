# Course Exam

## Workflow

1. Confirm exam profile: date, scope, format, score goal, current score, allowed materials, and grading rubric.
2. Diagnose the score gap by topic, question type, timing, accuracy, and expression quality.
3. Prioritize high-yield topics and likely point gains.
4. Create a timed practice and mock exam schedule.
5. Add test-taking strategy and final 24-hour plan.
6. Route wrong answers to `course-mistake-notebook`.

## Exam Rules

- If exam scope or format is unknown, state assumptions and ask for syllabus/sample paper.
- Balance high-frequency basics with the learner's weakest scoring areas.
- Include timing strategy for exams with strict time limits.
- For essay or language exams, include rubric-based revision and model outlines.
- For open-book university exams, focus on concept access, index sheets, application, and synthesis.

## Output

Return:

- Exam profile.
- Score gap diagnosis.
- Priority topic list.
- Practice schedule.
- Mock exam plan.
- Test-day strategy.
- Final 24-hour checklist.
