# Course Mistake Notebook

## Workflow

1. Capture the original question, learner answer, correct answer, rubric, and score if available.
2. Classify the mistake: concept, method, reading, calculation, expression, memory, timing, carelessness, or strategy.
3. Diagnose root cause, not only the correct answer.
4. Rewrite the solution or response in a reusable pattern.
5. Generate 2-5 transfer drills.
6. Schedule rechecks and route persistent gaps to tutoring or review.

## Mistake Rules

- A mistake entry must include "why I made it" and "how I will catch it next time".
- For language/writing errors, include evidence, wording, structure, and rubric criteria.
- For STEM errors, include decision cue, formula/definition, steps, and verification method.
- Track repeated errors as patterns, not isolated accidents.

## Output

Return:

- Mistake table.
- Root-cause diagnosis.
- Correct solution pattern.
- Transfer drills.
- Recheck schedule.
- Summary of repeated error patterns.
