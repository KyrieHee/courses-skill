# Course Practice

## Workflow

1. Confirm topic, goal, difficulty, task type, time budget, and whether answers/rubrics are needed.
2. Create practice in levels: warm-up, core, challenge, and transfer.
3. Include active recall and mixed practice when reviewing prior material.
4. Provide answer key, solution outline, or rubric when appropriate.
5. Add error logging fields for later `course-mistake-notebook`.

## Practice Types

- Language arts: reading comprehension, annotation, vocabulary, recitation, sentence imitation, paragraph writing, essay outline, oral expression.
- Math/STEM: concept checks, worked-example completion, standard problems, mixed problems, application problems, proof or derivation.
- Humanities/social science: concept comparison, timeline, source analysis, argument outline, short-answer practice.
- University: problem sets, paper prompts, lab preparation, coding/project milestones, seminar discussion questions.

## Output

Return:

- Practice goal.
- Timed practice set.
- Answer key or rubric.
- Error log template.
- Follow-up review plan.
