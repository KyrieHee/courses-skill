# Course Synthesis

## Workflow

1. Consolidate course profile, diagnosis, preview, tutoring, practice, review, exam, mistake, and reflection artifacts.
2. Identify the learner's main objective and limiting bottleneck.
3. Prioritize tasks by urgency, learning impact, confidence, and available time.
4. Build a calendar plan with checkpoints and evidence outputs.
5. Add risk controls and adjustment rules.
6. End with the first action to take today.

## Synthesis Rules

- Make the plan executable: each task should fit a realistic study block.
- Separate daily work, weekly review, exam sprint, and long-term habits.
- Use tables for schedules and concise bullets for judgment.
- Include what to stop doing when the plan is overloaded.
- For young learners, make parent/teacher support explicit but not controlling.
- For university learners, include syllabus alignment, office hours, assignment milestones, and grading weights when known.

## Output

Return:

- Summary judgment.
- Goals and success metrics.
- Priority topics and bottlenecks.
- 7-day or multi-week schedule.
- Stage tasks and handoffs.
- Checkpoints and adjustment rules.
- Today's first action.
